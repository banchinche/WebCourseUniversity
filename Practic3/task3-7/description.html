Описание

Здесь можно рассчитывать зарплату для сотрудников!

Сервис предоставляет такие функции:

  1. Предоставляемый общий функционал:
    1. Подсчитать сумму выплат заданному сотруднику за указанный диапазон месяцев и за каждый месяц в отдельности
    2. Подсчитать среднюю сумму выплат сотрудникам с указанной должностью за указанный диапазон месяцев и найти тех сотрудников, которые получают больше среднего месячного заработка
    3. Для заданного сотрудника проверить, не превышает ли за заданный месяц сумма его выплаты двойного оклада, и, если этот факт имеет место, то выдать соответствующее сообщение
    4. Вывести ведомость (табельный номер, фамилию, сумму выплат со всеми надбавками) на выплату зарплаты за указанный месяц, упорядоченную по фамилиям сотрудников
    5. Найти сотрудников, которым никогда не назначали заданную надбавку, а также надбавки, которые не назначали никогда никому за заданный диапазон месяцев
  2. Доступные операции для сотрудников
    1. Создавать
    2. Получить анкетные данные, табельный номер, должность, размер основного оклада
    3. Переводить на новую должность
    4. Начислять премии
    5. Вычитать налоги
